## 2024-05-18 - [Tar Slip Vulnerability in HTTP Fallback Tarball Extraction]
**Vulnerability:** A path traversal vulnerability was present in `lib/rulepack/source.rb`'s `extract_tar_gz` method. It failed to restrict extracted paths to the destination directory due to unsafe reliance on string manipulation (`parts[1..-1].join('/')`) rather than verifying the resolved absolute path.
**Learning:** Even though the HTTP fallback extraction routine skips the top-level directory and splits the path using `/`, path traversal payloads with `..` can bypass these naive checks and create files outside the temporary git clone fallback directory.
**Prevention:** Always use `File.expand_path` to resolve the final destination path and verify that it strictly starts with the expanded destination directory path (`dest_path.start_with?(expanded_dest_dir + File::SEPARATOR)`) before creating directories or files.
