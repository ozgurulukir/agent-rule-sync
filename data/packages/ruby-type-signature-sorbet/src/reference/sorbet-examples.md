This file documents real-world Sorbet RBI patterns. The full example set is
distributed via the upstream `ruby-agent-skills` repository and mirrors the
`reference/rbs_by_example.md` structure from the generating-rbs skill.

## Using the Examples

The examples follow standard Sorbet RBI conventions (see `syntax.md`).
Apply patterns from these files when generating RBI signatures for
production Ruby gems.
